# Flashcards App

Statisk flashcard-app til IB prep.

## Lokal struktur

- `index.html`: app-shell
- `styles.css`: styling
- `app.js`: flashcard-logik og data
- `Pictures/` og `LinkedIN/`: lokale assets

## Deployment

Appen kan deployes direkte som statisk site på Vercel uden build step.

## Anbefalet workflow

1. Rediger filer lokalt.
2. Commit og push til GitHub.
3. Vercel bygger og opdaterer live-sitet automatisk fra GitHub.
